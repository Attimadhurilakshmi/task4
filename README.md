This project is part of the Task-4: Security Enhancements implementation for a web application. 
The objective of this task is to improve the security of the application by implementing proper validation and secure login functionality.

The project includes a secure login system built using HTML, CSS, and Node.js (Express).
It features a clean user interface with a bordered login form and a green login button for better user experience.

Key features implemented in this project include:

Client-side form validation to ensure all required fields are filled before submission.
Server-side validation using Node.js to verify user credentials securely.
Role-based access control for different users such as Admin and User.
Improved user interface design using CSS with borders and styled buttons.
Secure login handling to prevent invalid login attempts.

Technologies used in this project:

HTML
CSS
Node.js
Express.js

This project demonstrates how security measures like validation and controlled user access can be integrated into a web application 
to protect user data and improve reliability.
